# Editor Markdown + Git (sem terminal)

Fluxo simples para editar notas e publicar sem terminal.

## Passos

1. Instale o GitHub Desktop.
2. Clone seu repositorio.
3. Abra a pasta no seu editor.
4. Edite arquivos em src/pages.
5. Commit e Push no GitHub Desktop.

## Dicas

- Arquivos com _ sao ignorados.
- Imagens ficam em src/assets.
